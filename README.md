# DNN-GWAS (DNN-aided genetic association testing)
Source code for the proposed method in "Deep neural network-aided genetic association testing in samples with related individuals"

funEL.R: all functions needed for generating simulated data, performing likelihood score (LS) and quasi-likelihood score (QLS) tests, and constructing different DNNs

Sim_Con_H0.R: code for checking consistency among LRT/QLS/DNN-LRT/DNN-QLS/DNN-LRT-RAW/DNN-QLS-RAW under H0

Sim_Con_Ha.R: code for checking consistency among LRT/QLS/DNN-LRT/DNN-QLS/DNN-LRT-RAW/DNN-QLS-RAW under Ha

Sim_Eva_All_H0.R: code for evaluating LRT/QLS/DNN-LRT/DNN-QLS/DNN-ENS/DNN-LRT-RAW/DNN-QLS-RAW/DNN-LRT-AF/DNN-QLS-AF/DNN-ENS-AF under H0

Sim_Eva_All_Ha.R: code for evaluating LRT/QLS/DNN-LRT/DNN-QLS/DNN-ENS/DNN-LRT-RAW/DNN-QLS-RAW/DNN-LRT-AF/DNN-QLS-AF/DNN-ENS-AF under Ha
